# LucidByte --- Полноценный Реверс-Инженер Android / Анализ Вредоносного ПО

## Описание

**LucidByte** --- это комплексная система статического и динамического
анализа безопасности Android‑приложений с использованием Large Language
Model. Система обнаруживает подозрительные разрешения, опасные
API-вызовы, сигнатуры вредоносного ПО, анализирует нативный код,
выявляет упаковщики и выполняет динамический анализ поведения
приложения.

------------------------------------------------------------------------

## Возможности

### Статический анализ

-   Декомпиляция APK, JAR, DEX
-   Анализ разрешений Android
-   Сканирование опасных API
-   Обнаружение сигнатур вредоносного ПО
-   Анализ нативных библиотек (.so)
-   Обнаружение упаковщиков и обфускаторов
-   Автоматическая деобфускация

### Динамический анализ

-   Запуск приложения в Android‑эмуляторе
-   Frida‑хуки для перехвата вызовов
-   Перехват трафика через Mitmproxy
-   Мониторинг поведения в реальном времени
-   Анализ криптографии

### Искусственный интеллект

-   Семантический анализ кода
-   Переименование обфусцированных методов
-   Генерация отчётов безопасности
-   Объяснение логики работы методов

### Визуализация

-   Графический интерфейс
-   Граф вызовов методов
-   Карта разрешений и рисков
-   Монитор сетевого трафика

------------------------------------------------------------------------

## Требования

### Аппаратные

-   Windows 10/11 или Linux
-   32 GB RAM (рекомендуется)
-   NVIDIA GPU с CUDA (опционально)
-   50 GB свободного места

### Программные

-   Python 3.9+
-   Android SDK + Emulator
-   jadx
-   Ghidra
-   Ollama + модель qwen2.5-coder:7b
-   ADB

------------------------------------------------------------------------

## Установка

``` bash
pip install -r requirements.txt
```

### Установка инструментов

-   JADX: https://github.com/skylot/jadx/releases
-   Ghidra: https://github.com/NationalSecurityAgency/ghidra/releases
-   Ollama: https://ollama.com

``` bash
ollama run qwen2.5-coder:7b
```

### Создание Android эмулятора

``` bash
avdmanager create avd -n Pixel_4_API_30 -k "system-images;android-30;google_apis;x86"
```

### Запуск

``` bash
python run.py
```

------------------------------------------------------------------------

## Предупреждение

Анализ вредоносного ПО должен выполняться **только в изолированной
среде** (виртуальная машина). Не запускайте подозрительные APK на
основной системе.

------------------------------------------------------------------------

## Структура отчёта

Система формирует отчёт:

1.  Общий уровень риска (1--10)
2.  Статический анализ
3.  Найденные сигнатуры
4.  Упаковщики
5.  Нативный код
6.  Динамический анализ
7.  Сетевой трафик
8.  Заключение LLM
